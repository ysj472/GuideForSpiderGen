# 06  화면에 WebPage 추가

WebView\
웹 문서를 애플리케이션 내의 특정 컴포넌트 영역에 로드하여 표시할 수 있는 컴포넌트.

WebView를 사용하면 웹 페이지를 애플리케이션의 일부로 통합할 수 있으며, 다양한 웹 콘텐츠를 표시.

자세한 구현은 [링크](https://wikidocs.net/24839) 참조
